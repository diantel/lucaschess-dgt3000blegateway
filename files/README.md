# Packaged Files

These files are installed into Lucas Chess by `install.ps1`.

- `Eboard.py`: patched Lucas Chess electronic-board integration.
- `Eboard.patch`: maintainer note/diff instructions for reviewing the Lucas patch.
- `dgt3000_ble_sidecar.py`: BLE sidecar for Tortue95's DGT3000 gateway.

Do not edit the installed Lucas files manually unless you also update these packaged copies.
